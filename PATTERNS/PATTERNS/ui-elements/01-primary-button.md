# Primary Button

> Pattern · UI Elements · stable

The standard primary call-to-action button — amber fill, rounded-xl, press-scale on click, accessible focus ring. Reuse it for every primary action; extend the className instead of re-deriving the base styles.

## Source code example

```
<button className="inline-flex items-center gap-2 h-11 px-6 rounded-xl bg-primary text-primary-foreground text-sm font-semibold hover:opacity-90 active:scale-95 transition-all shadow-lg shadow-primary/25">{children}</button>
```
