## Why

Возможность набрасывать заметки голосом или текстом прямо в Telegram в свободной форме,
без необходимости открывать приложение или отдельный инструмент. Сохранённые заметки доступны
через диалог — «напомни мне» запускает семантический поиск по всей истории.

## Early ТЗ (голосовая диктовка владельца, 2026-07-05)

**Два триггера в Telegram:**

**«кстати говоря запомни это:»** — всё после этой фразы сохраняется в историю (векторная база LightRAG).
Формат свободный: заметка, идея, факт. Инициирует RAG-ingest.

**«кстати говоря напомни мне:»** — запускает поиск по ранее сохранённым записям. Автоматизация ищет
семантически близкие совпадения в векторной базе и возвращает ответ пользователю в режиме диалога
прямо в Telegram.

## How it works

| Узел | Kind | Задача |
|------|------|--------|
| Telegram webhook | trigger | Принимает входящее сообщение |
| Phrase router | router | Определяет триггер: «запомни» или «напомни» |
| Save note | action | Ingest текста после «запомни это:» в LightRAG |
| Search memory | action | Query LightRAG по тексту после «напомни мне:» |
| Reply | action | Отправляет ответ пользователю в Telegram |

## Efficiency

Все процессы на локальном VPS — нет внешних API-вызовов кроме Telegram Bot API.
LightRAG и Hermes уже работают в субстрате, дополнительного деплоя не требуется.

## Reuse

- `fractera-bridge` (Telegram webhook) переиспользуется любой другой автоматизацией с Telegram.
- RAG-ingest паттерн применим к любому источнику коротких записей.

## Result

Пользователь получает работающую пару команд в личном Telegram-боте:
одна сохраняет заметку, вторая ищет по истории заметок и отвечает в диалоге.
